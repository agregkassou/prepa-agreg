\section{Les trois modes de transfert thermiques}
\subsection{Notion de bilan thermique}
\begin{itemize}
\item partir du premier principe
\item écrire $Q$ en fonction de $\vect{j}\e{th}$
\item préciser les travaux reçus en particulier le terme source (on insistera sur leur différentes natures : chimique, nucléaire...)
\item intégrer pour obtenir un bilan global
\item donner le cas stationnaire
\end{itemize}

\subsection{La conduction [2]}
\subsubsection{Rappels}
\begin{itemize}
\item vecteur densité de courant
\item équation de conservation
\item loi de Fourier
\item équation de la chaleur
\item insister sur les ODG de $D$
\item temps et distances caractéristiques de diffusion / ODG associés
\end{itemize}

\subsubsection{Notion de résistance thermique}
\begin{itemize}
\item on se place en régime permanent : $\Delta T=0$
\item faire la démonstration dans le cadre de la conduction à travers un mur
\item exprimer $R\e{th}$
\item faire l'analogie électrique
\item vérifier la loi d'additivité série (plusieurs murs collés)
\item donner sans démonstration la loi d'additivité parallèle (mur et fenêtre)
\end{itemize}

\subsection{La convection}
\subsubsection{Définition [3]}
\begin{itemize}
\item définir convection
\item distinguer convection naturelle et forcée 
\item \underline{manipulation} : tube en O chauffé à la base (convection naturelle) et agitation d'une tasse (convection forcée)
\item en toute généralité, c'est un problème complexe d'hydrodynamique : il faut introduire les flux d'énergie et de matière et étudier le couplage entre les deux
\item on va donc simplifier le problème!
\end{itemize}

\section{Illustration du couplage entre les modes de transfert}
\subsection{Position du problème [1]}
\begin{itemize}
\item on considère une ailette de refroidissement
\item bilan en l'absence d'ailette
\item \textbf{ajouter le rayonnement (non considéré dans le problème)}. L'expression peut être linéarisée autour de la température moyenne car la différence entre les deux températures est faible. Finalement on obtient un $h$ effectif modifié par le terme de rayonnement
\item on ajoute une ailette
\item présenter les simplifications associées
\item bien indiquer les trois types de transfert mis en jeu dans le problème
\end{itemize}

\subsection{Résolution [1]}
\begin{itemize}
\item bilan thermique sur une tranche de l'ailette (toujours avec un $h$ effectif)
\item en déduire $T(z)$
\item conditions limites à expliciter
\item aller vite sur le calcul un peu lourd
\item courbe de température en fonction de $L$
\item calcul d'efficacité $\eta$ et limite ailette longue
\item interpréter tous les coefficients qui apparaissent dans $\eta$
\item un bon refroidissement nécessite : grande surface de contact avec le fluide, convection forcée, ailettes peintes en noir
\end{itemize}
