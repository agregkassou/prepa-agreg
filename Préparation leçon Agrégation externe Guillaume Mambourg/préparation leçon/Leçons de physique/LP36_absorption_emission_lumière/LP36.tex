\subsection{Comment faire un laser? [3]}
\begin{itemize}
\item la description du Aspect est un peu trop complète (mais claire): ne conserver que les grandes lignes
\item il faut être clair et précis : milieu à gain mais de gain faible, alors il faut reboucler pour faire de nombreux passages dans le milieu amplificateur
\item cavité (avec faible perte pour sortir un faisceau quand même)
\item cette cavité donne les modes longitudinaux possibles (séparés de l'ISL)
\item faire une présentation sous forme de système bouclé avec gain et perte sur un tour
\item un mode peut exister si pour ce mode, le gain est supérieur aux pertes
\item \underline{manipulation} : présenter en parallèle sur le laser en kit qu'on fera fonctionner en direct
\item parler du cas des lasers monomodes, multimodes... on pourra aussi parler de saut entre modes
\end{itemize}
